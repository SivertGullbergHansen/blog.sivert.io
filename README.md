## Installation

### Clone the project
```sh
git clone https://github.com/SivertGullbergHansen/blog.sivert.io.git blog
cd blog
```

### Install dependencies
```sh
npm i
```
or
```sh
yarn
```

### Run development server
```sh
npm run dev
```
or
```sh
yarn dev
```
